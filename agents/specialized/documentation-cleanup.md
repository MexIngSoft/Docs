# Agent especializado Documentation Cleanup

## Alcance

Indices, agents, mapas, estructura documental, archivos temporales y limpieza.

## Reglas

- No borrar documentos fuente.
- No mover ni eliminar `AGENTS-*.md`.
- Limpiar agent completado significa vaciar contenido y conservar archivo.
- No reescribir por estilo si no cambia decision, estado o ruta.

## Validacion

- Rutas mencionadas existen.
- No se crean duplicados canonicos.
