## ¿Cómo se relaciona con el ecosistema?

Aquí es donde veo algo muy interesante.

En lugar de competir con tus otros proyectos, Comerzy se convierte en la puerta de entrada.

```
Persona vende velas
        │
        ▼
     Comerzy
        │
Crece su negocio
        │
Necesita inventario
        ▼
 Universal POS
        │
Abre segunda sucursal
        ▼
    JobCron ERP
        │
Quiere vender en Amazon
        ▼
Master Marketplace
        │
Necesita una tienda propia
        ▼
MexIngSof Web
```

Lo mismo ocurre con otros tipos de negocios:

* Una refaccionaria local comienza en Comerzy para vender en su ciudad. Cuando quiere vender en todo México, migra a Refapart y puede apoyarse en Master Marketplace.
* Una tienda de regalos empieza con un catálogo sencillo. Cuando requiere controlar inventario y sucursales, incorpora Universal POS y después JobCron.
* Un restaurante puede iniciar con pedidos locales y, si crece, adoptar herramientas más avanzadas del ecosistema.

Eso significa que **Comerzy no es un producto final**.

Es **el primer escalón**.

No busca quedarse con el cliente para siempre.

Busca ayudarlo a crecer hasta que necesite herramientas más potentes dentro del mismo ecosistema.

Desde una perspectiva de negocio, eso crea un recorrido muy claro: primero ayudas al comercio local a vender más cerca de donde está; cuando su operación deja de ser local, ya tienes preparadas las siguientes soluciones para acompañarlo en esa nueva etapa, sin obligarlo a cambiar de proveedor ni de plataforma.


Sí, ya leí el repo **MexIngSoft/Docs** en rama `general`. La regla central es clara: **Frontend → Gateway General → APIs compartidas → API especializada solo si aplica**; además, no se debe crear una API nueva si una compartida ya cubre la responsabilidad.

## Decisión para Comerzy

**Comerzy sí necesita API especializada**, pero pequeña.

No debe duplicar catálogo, inventario, precios, ventas, direcciones, notificaciones ni auth, porque esos dominios ya están definidos como APIs reutilizables en la documentación.

### APIs que se reutilizan

| Necesidad Comerzy           | API correcta            |
| --------------------------- | ----------------------- |
| Usuarios, login, permisos   | `Auth API`              |
| Entrada web/app             | `Gateway General`       |
| Productos locales           | `Catalog API`           |
| Stock                       | `Inventory API`         |
| Precios/promos              | `Pricing API`           |
| Pedidos                     | `Sales API`             |
| Direcciones/zonas           | `Address API`           |
| Notificaciones              | `Notification API`      |
| Activar módulos por cliente | `FeatureVisibility API` |
| Cargas masivas              | `ETL API`               |

### APIs nuevas necesarias

| API nueva                 | Tipo              | Por qué                                                                                                                 |
| ------------------------- | ----------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `API.PY.DJANGO.Comerzy`   | Especializada     | Reglas de comercio local: radio de venta, cobertura por zona, comercios locales, horarios, visibilidad local.           |
| `API.PY.DJANGO.Logistics` | Core/shared nueva | La logística compartida no es exclusiva de Comerzy; también aplica a Refapart, TecnoTelec, JobCron y futuros proyectos. |

La documentación menciona logística como parte del núcleo ERP reutilizable, pero en el mapa de APIs todavía no aparece como API formal.  Por eso yo la agregaría como **API compartida**, no como parte de Comerzy.

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass
from decimal import Decimal


@dataclass
class DeliveryQuote:
    provider: str
    amount: Decimal
    currency: str
    expires_at: str | None = None
    raw: dict | None = None


@dataclass
class DeliveryResult:
    provider: str
    external_delivery_id: str
    status: str
    tracking_url: str | None
    raw: dict


class DeliveryProviderClient(ABC):
    @abstractmethod
    def quote(self, payload: dict) -> DeliveryQuote:
        pass

    @abstractmethod
    def create_delivery(self, payload: dict) -> DeliveryResult:
        pass

    @abstractmethod
    def get_delivery(self, external_delivery_id: str) -> DeliveryResult:
        pass

    @abstractmethod
    def cancel_delivery(self, external_delivery_id: str) -> DeliveryResult:
        pass
```

```python
# apps/logistics/providers/uber_direct.py

import requests
from decimal import Decimal
from .base import DeliveryProviderClient, DeliveryQuote, DeliveryResult


class UberDirectClient(DeliveryProviderClient):
    def __init__(self, base_url: str, token: str, customer_id: str):
        self.base_url = base_url.rstrip("/")
        self.token = token
        self.customer_id = customer_id

    def _headers(self):
        return {
            "Authorization": f"Bearer {self.token}",
            "Content-Type": "application/json",
        }

    def quote(self, payload: dict) -> DeliveryQuote:
        url = f"{self.base_url}/customers/{self.customer_id}/delivery_quotes"
        response = requests.post(url, json=payload, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()

        return DeliveryQuote(
            provider="uber_direct",
            amount=Decimal(str(data.get("fee", 0))) / 100,
            currency=data.get("currency", "MXN"),
            raw=data,
        )

    def create_delivery(self, payload: dict) -> DeliveryResult:
        url = f"{self.base_url}/customers/{self.customer_id}/deliveries"
        response = requests.post(url, json=payload, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()

        return DeliveryResult(
            provider="uber_direct",
            external_delivery_id=data["id"],
            status=data.get("status", "created"),
            tracking_url=data.get("tracking_url"),
            raw=data,
        )

    def get_delivery(self, external_delivery_id: str) -> DeliveryResult:
        url = f"{self.base_url}/customers/{self.customer_id}/deliveries/{external_delivery_id}"
        response = requests.get(url, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()

        return DeliveryResult(
            provider="uber_direct",
            external_delivery_id=data["id"],
            status=data.get("status", "unknown"),
            tracking_url=data.get("tracking_url"),
            raw=data,
        )

    def cancel_delivery(self, external_delivery_id: str) -> DeliveryResult:
        url = f"{self.base_url}/customers/{self.customer_id}/deliveries/{external_delivery_id}/cancel"
        response = requests.post(url, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()

        return DeliveryResult(
            provider="uber_direct",
            external_delivery_id=external_delivery_id,
            status="cancelled",
            tracking_url=data.get("tracking_url"),
            raw=data,
        )
```

```python
# apps/logistics/providers/skydropx.py

import requests
from decimal import Decimal
from .base import DeliveryProviderClient, DeliveryQuote, DeliveryResult


class SkydropxClient(DeliveryProviderClient):
    def __init__(self, base_url: str, api_key: str):
        self.base_url = base_url.rstrip("/")
        self.api_key = api_key

    def _headers(self):
        return {
            "Authorization": f"Token token={self.api_key}",
            "Content-Type": "application/json",
        }

    def quote(self, payload: dict) -> DeliveryQuote:
        url = f"{self.base_url}/v1/quotations"
        response = requests.post(url, json=payload, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()

        amount = data.get("total") or data.get("amount") or 0

        return DeliveryQuote(
            provider="skydropx",
            amount=Decimal(str(amount)),
            currency=data.get("currency", "MXN"),
            raw=data,
        )

    def create_delivery(self, payload: dict) -> DeliveryResult:
        url = f"{self.base_url}/v1/shipments"
        response = requests.post(url, json=payload, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()

        shipment = data.get("shipment", data)

        return DeliveryResult(
            provider="skydropx",
            external_delivery_id=str(shipment.get("id")),
            status=shipment.get("status", "created"),
            tracking_url=shipment.get("tracking_url"),
            raw=data,
        )

    def get_delivery(self, external_delivery_id: str) -> DeliveryResult:
        url = f"{self.base_url}/v1/shipments/{external_delivery_id}"
        response = requests.get(url, headers=self._headers(), timeout=20)
        response.raise_for_status()
        data = response.json()
        shipment = data.get("shipment", data)

        return DeliveryResult(
            provider="skydropx",
            external_delivery_id=external_delivery_id,
            status=shipment.get("status", "unknown"),
            tracking_url=shipment.get("tracking_url"),
            raw=data,
        )

    def cancel_delivery(self, external_delivery_id: str) -> DeliveryResult:
        url = f"{self.base_url}/v1/cancel_label_requests"
        response = requests.post(
            url,
            json={"shipment_id": external_delivery_id},
            headers=self._headers(),
            timeout=20,
        )
        response.raise_for_status()
        data = response.json()

        return DeliveryResult(
            provider="skydropx",
            external_delivery_id=external_delivery_id,
            status="cancel_requested",
            tracking_url=None,
            raw=data,
        )
```

```python
    if provider == "uber_direct":
        return UberDirectClient(
            base_url=credentials["base_url"],
            token=credentials["token"],
            customer_id=credentials["customer_id"],
        )

    if provider == "99minutos":
        return Minutos99Client(
            base_url=credentials["base_url"],
            jwt_token=credentials["jwt_token"],
        )

    if provider == "skydropx":
        return SkydropxClient(
            base_url=credentials["base_url"],
            api_key=credentials["api_key"],
        )

    raise ValueError(f"Unsupported logistics provider: {provider}")
```

# Endpoint interno de Logistics API

```python
# apps/logistics/views.py

from rest_framework.views import APIView
from rest_framework.response import Response
from rest_framework import status

from .models import DeliveryRequest, LogisticsCredential
from .providers.factory import build_delivery_provider


class CreateDeliveryView(APIView):
    def post(self, request):
        tenant_id = request.headers.get("X-Tenant-Id")
        provider = request.data["provider"]

        credential = LogisticsCredential.objects.get(
            tenant_id=tenant_id,
            provider=provider,
            is_active=True,
        )

        client = build_delivery_provider(
            provider=provider,
            credentials=credential.encrypted_credentials,
        )

        result = client.create_delivery(request.data["payload"])

        delivery = DeliveryRequest.objects.create(
            tenant_id=tenant_id,
            sales_order_id=request.data["sales_order_id"],
            provider=provider,
            status=result.status,
            pickup_address_id=request.data["pickup_address_id"],
            dropoff_address_id=request.data["dropoff_address_id"],
            external_delivery_id=result.external_delivery_id,
            tracking_url=result.tracking_url,
            raw_request=request.data,
            raw_response=result.raw,
        )

        return Response(
            {
                "id": str(delivery.id),
                "provider": provider,
                "external_delivery_id": result.external_delivery_id,
                "status": result.status,
                "tracking_url": result.tracking_url,
            },
            status=status.HTTP_201_CREATED,
        )
```

# Flujo para que un usuario se conecte rápido

1. Se registra en Comerzy usando `Auth`.
2. Crea su comercio local.
3. Define dirección, ciudad y radio de venta.
4. Sube productos.
5. Comerzy manda productos a `Catalog`.
6. Stock va a `Inventory`.
7. Precios van a `Pricing`.
8. Pedidos van a `Sales`.
9. Entrega va a `Logistics`.
10. Notificaciones van a `Notification`.

El comercio no configura todo desde cero. Debe poder iniciar con:

```text
Nombre del negocio
Tipo de producto
Dirección
Radio de venta
Horario
Productos iniciales
Método de entrega:
- Recolección
- Entrega local compartida
- Entrega propia
```
