## FASE 9 — Boceto PDF de JobCron

Buscar el archivo en:

archivo/images/jobcron/

Si existe PDF:

1. Extraer páginas como imágenes.
2. Guardarlas en:
   02_projects/jobcron/ui/source/

3. Crear versiones limpias en:
   02_projects/jobcron/ui/clean/

4. Crear documentación:
   02_projects/jobcron/ui/00_jobcron_mockup_analysis.md

Por cada página documentar:

- número de página
- nombre propuesto de pantalla
- módulo ERP relacionado
- componentes visibles
- botones
- tablas
- filtros
- formularios
- navegación
- datos requeridos
- API que alimenta la vista
- si requiere placeholder
- mejoras visuales
- elementos que deben eliminarse
- elementos que deben conservarse

Si el PDF no existe, escribir:
“No se encontró PDF de JobCron en archivo/images/jobcron/”.

Regla visual:
El boceto no es diseño final.
Es referencia funcional.
Debe limpiarse, reinterpretarse y llevarse a componentes modernos.
