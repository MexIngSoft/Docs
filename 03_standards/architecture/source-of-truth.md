# Fuente de verdad arquitectonica

Orden de precedencia:

1. `03_standards/global-governance.md` y estandares fechados vigentes.
2. Arquitectura canonica en `01_core_erp/architecture`.
3. Contratos y decisiones del proyecto en `02_projects`.
4. Agents activos.
5. Documentos historicos y `_archive`.

Un agent no puede revertir una decision canonica posterior. Actualmente la
entrada oficial es el Gateway central modular; las menciones a crear un Gateway
por proyecto son historicas o contradictorias y no gobiernan implementacion.
