# Politica de auditoria

## Eventos auditables

- Login/logout.
- Cambios de permisos.
- Cambios de precio/costo.
- Procesamiento de documentos.
- Descargas sensibles.
- Jobs fallidos o reintentados.

## Campos minimos

```text
actor
action
resource
resource_id
occurred_at
correlation_id
metadata
```
