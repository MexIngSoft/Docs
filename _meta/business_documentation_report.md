# Reporte de documentacion por proyecto

## Ruta canonica

El repositorio usa:

```text
Docs/02_projects/<proyecto>
```

No se crea `Docs/business` para evitar duplicidad.

## Reglas

- Cada proyecto debe documentar solo reglas propias.
- Los modulos reutilizables deben enlazarse a `01_core_erp`.
- Integraciones externas van en `04_integrations`.
