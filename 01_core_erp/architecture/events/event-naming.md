# Naming de eventos

## Formato

```text
NounPastTense
```

Ejemplos:

- `FileUploaded`
- `FileProcessingCompleted`
- `ProductPublished`
- `QuoteRequested`

## Reglas

- Usar ingles tecnico estable.
- No incluir nombres de empresas salvo eventos especificos de proyecto.
- Versionar payload cuando cambie contrato.
