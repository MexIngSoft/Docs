# Flujo de licitaciones publicas

```text
Registrar o importar oportunidad
Identificar empresa candidata
Clasificar categoria y alcance
Rules Engine evalua viabilidad inicial
Catalog y Supplier validan oferta posible
Pricing calcula costos y margen
Logistics estima entrega o instalacion
Projects arma solucion si aplica
Sales genera cotizacion/propuesta economica
Documents concentra anexos
Equipo decide participar
Se da seguimiento al resultado
Si se gana, se convierte a proyecto o venta
```

## Reglas

- El ERP debe iniciar como inteligencia y seguimiento.
- La participacion oficial debe hacerse conforme a la plataforma y requisitos vigentes.
- Cada empresa debe tener perfil de licitacion antes de participar.
