# Quote API

## Responsabilidad

Crear cotizaciones, partidas, snapshots, vigencias y aprobaciones.

## Endpoints minimos

- `POST /quotes`
- `GET /quotes/{id}`
- `PATCH /quotes/{id}`
- `POST /quotes/{id}/approve`
- `POST /quotes/{id}/expire`
- `POST /quotes/{id}/convert-to-order`
