# REFAPART Supplier Management

## Objetivo

Gestionar proveedores verificados para abastecimiento interno sin publicar su
contacto directo al cliente.

## Datos minimos

| Campo | Estado |
|---|---|
| BusinessName | PENDIENTE |
| SupplierType | PENDIENTE |
| WhatsappPhone | PENDIENTE |
| Phone | PENDIENTE |
| City | PENDIENTE |
| State | PENDIENTE |
| AddressText | PENDIENTE |
| BrandsJson | PENDIENTE |
| CategoriesJson | PENDIENTE |
| OpeningHoursJson | PENDIENTE |
| InternalRating | PENDIENTE |
| AverageResponseTimeMinutes | PENDIENTE |
| ReliabilityScore | PENDIENTE |
| IsAuthorized | PENDIENTE |
| IsActive | PENDIENTE |
| Notes | PENDIENTE |

## Reglas

- Proveedor autorizado no equivale a proveedor visible para cliente.
- Proveedor bloqueado no participa en sugerencias.
- Cambios de estado deben quedar auditados.
