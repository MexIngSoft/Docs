# REFAPART Warranty And Returns Policy

## Objetivo

Definir reglas iniciales para garantia, devoluciones e incidencias de piezas.

## Politica inicial

| Tema | Decision |
|---|---|
| Garantia minima | PENDIENTE_DE_DEFINIR |
| Piezas usadas | Requieren condicion, fotos y garantia visible si aplica. |
| Piezas nuevas | Requieren proveedor autorizado o evidencia de origen. |
| Piezas reparadas | Deben indicarse como reparadas antes de cotizar. |
| Devolucion | PENDIENTE_DE_DEFINIR |
| Cancelacion | PENDIENTE_DE_DEFINIR |
| Incidencia logistica | Registrar evidencia y estado. |

## Regla

No prometer garantia, devolucion o disponibilidad si no existe evidencia del
proveedor o politica aprobada.
