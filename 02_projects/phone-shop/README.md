# Proyecto Phone Shop

## Rol

Canal/proyecto especializado en telefonia, accesorios y servicios relacionados.

## Que puede mostrar

- Telefonos.
- Accesorios.
- Cables.
- Cargadores.
- Fundas.
- Equipos reacondicionados si aplica.
- Reparacion, configuracion y soporte.

## Relacion con ERP

Usa:

- Catalog.
- Supplier.
- Pricing.
- Inventory.
- Sales.
- Cart.
- POS.
- ChannelAssortment.

## Diferencia con Tecno Telec

Lo que en Tecno Telec puede verse mal como venta generica de telefonos, en Phone Shop puede ser producto principal.
