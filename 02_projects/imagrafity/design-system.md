# Sistema de diseno Imagrafity

## Concepto

Imagrafity combina arte, tecnologia y personalizacion de productos. La identidad
debe sentirse creativa, energetica y digital, sin perder claridad comercial.

Frase base:

```text
La imaginacion se imprime.
```

## Paleta base

| Rol | Color | HEX | Uso |
|---|---|---|---|
| Primario | Azul electrico | `#00E5FF` | Acentos, botones principales, highlights. |
| Secundario | Fucsia neon | `#FF007C` | Llamados creativos, hover, energia visual. |
| Apoyo | Violeta digital | `#9B51E0` | Gradientes, sombras y transiciones. |
| Fondo | Negro grafito | `#0F0F0F` | Fondos hero y editor. |
| Superficie | Gris titanio | `#1A1A1A` | Paneles y secciones oscuras. |
| Texto claro | Blanco humo | `#F5F5F5` | Texto sobre fondos oscuros. |
| Exito | Verde menta | `#00FFB3` | Confirmaciones y estados positivos. |

## Reglas visuales

- El producto personalizado debe ser protagonista.
- Usar imagenes reales o renders de producto, no solo tarjetas decorativas.
- El editor/preview debe aparecer temprano en la experiencia.
- Mantener contraste suficiente pese al uso de neon.
- No saturar pantallas operativas con efectos visuales.
- Reservar glassmorphism y neon para marca, hero, editor e inspiracion.

## Componentes clave

- Hero con producto visible.
- Catalogo por categorias.
- Editor de personalizacion.
- Preview 2D/3D.
- Galeria de inspiracion.
- Cotizador/checkout.
- Panel de usuario con pedidos y disenos.
- Comunidad o creadores.

## Estados obligatorios

- Producto sin diseno.
- Diseno cargado.
- Preview generandose.
- Error de archivo o formato.
- Cotizacion lista.
- Orden enviada a produccion.
