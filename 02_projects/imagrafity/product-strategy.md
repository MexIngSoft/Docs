# Estrategia de producto Imagrafity

## Objetivo

Convertir Imagrafity en una plataforma de personalizacion creativa, no solo en
una tienda de productos sublimados.

## Propuesta de valor

```text
Convertir ideas en productos visuales unicos usando personalizacion avanzada,
IA, diseno interactivo y produccion inteligente.
```

## Diferenciadores

- Editor visual simple.
- Preview de producto.
- Plantillas creativas.
- Inspiracion y comunidad.
- Produccion conectada a ordenes.
- Cotizacion automatica.
- Potencial B2B para empresas, eventos y creadores.

## MVP

1. Catalogo de productos personalizables.
2. Detalle de producto.
3. Editor basico.
4. Preview.
5. Prompts editables por producto.
6. Preview obligatoria sobre producto.
7. Cotizacion/compra.
8. Orden de produccion.
9. Panel de usuario.

## Expansiones

- Academia creativa.
- Portal B2B/distribuidores.
- Concursos o retos creativos.
- API de pedidos.
- Plantillas premium.
- Marketplace de disenos.
- Integracion con produccion e inventario.
- Editor 2D con Konva.js.
- Preview 3D con Three.js.
- Generacion IA conectada a prompts por producto.

## Relacion con ERP

| Necesidad | Responsable |
|---|---|
| Clientes, pagos y facturacion | Core ERP / Billing |
| Catalogo base | Core Catalog + Imagrafity API |
| Personalizacion | Imagrafity |
| Prompts y previews | Imagrafity |
| Produccion | Imagrafity + JobCron |
| Envios | Core Logistics |
| Reportes | JobCron / ERP |
