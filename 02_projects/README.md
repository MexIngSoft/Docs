# Proyectos

## Objetivo

Esta carpeta contiene la documentacion especifica de cada empresa, marca, producto o implementacion que puede usar el nucleo ERP JobCron.

## Regla

Si el contenido habla de identidad, servicios, segmento, frontend, marca, flujo comercial o prioridades particulares de una empresa, pertenece al proyecto correspondiente.

Si el contenido aplica a todos los proyectos, pertenece a `01_core_erp`.
