# MockWander

## Rol

Proyecto de privacidad/ubicacion para simular rutas.

## Repositorio base

```text
https://github.com/1CASH1/MockWander
```

## Que hace

Idea inicial: ocultar ubicacion en tiempo real o trazar rutas falsas.

## Nota de seguridad y alcance

Este proyecto requiere definicion cuidadosa para evitar usos indebidos. Debe orientarse a escenarios legitimos como privacidad personal, pruebas de aplicaciones, simulacion QA, demos, geofencing de desarrollo o entrenamiento.

No debe documentarse para evadir controles legales, engañar servicios, cometer fraude o burlar restricciones de plataformas.

## Necesidades JobCron

Puede ser independiente.

Opcional:

- Auth.
- Subscription.
- Analytics.
- Device management.

## MVP sugerido

1. Simulador de ruta para pruebas.
2. Perfiles de ubicacion.
3. Modo demo.
4. Exportacion de rutas.
