# Tip Smart

## Rol

Aplicacion en linea para calcular propinas.

## Dominios

PENDIENTE_DE_DEFINIR.

## Que hace

Calcula propinas, divide cuentas y puede guardar preferencias o historial si se decide.

## Necesidades JobCron

Puede iniciar independiente sin ERP completo.

Opcional:

- Auth.
- Analytics.
- Payments si agrega version premium.

## MVP sugerido

1. Calculadora de propina.
2. Division entre personas.
3. Ajustes por porcentaje.
