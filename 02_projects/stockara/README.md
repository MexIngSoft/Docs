# Stockara

## Rol

Inventarios y almacenes.

## Dominios

- stockara.com
- stockara.mx

## Que hace

Controla almacenes, stock, movimientos, reservas y auditoria.

## Necesidades JobCron

- Inventory.
- Warehouses.
- Stock movements.
- Reservations.
- Reports.

## MVP sugerido

1. Almacenes.
2. Inventario por almacen.
3. Movimientos.
4. Reservas.
