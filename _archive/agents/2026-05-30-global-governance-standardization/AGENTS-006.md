Sí, todavía falta definir una **estrategia de publicidad pasiva**, porque si dependes únicamente de anuncios recompensados, muchos usuarios nunca los verán y los ingresos serán bajos durante la etapa inicial de DocuCore.

Analizando todo lo que has planteado para DocuCore (herramientas documentales, enfoque empresarial, UX profesional, ERP de monetización y crecimiento hacia clientes empresariales), yo dividiría la publicidad en **4 niveles de intensidad**.

# Nivel 1: Publicidad pasiva permanente (recomendado)

No interrumpe el flujo.

Ubicaciones:

### Panel lateral derecho

Cuando el usuario trabaja con documentos.

```text
┌─────────────────────────────┬──────────────┐
│                             │              │
│      Vista previa PDF       │ Patrocinado  │
│                             │              │
│                             │ Syscom       │
│                             │ CT           │
│                             │ HP           │
│                             │ Canon        │
│                             │              │
└─────────────────────────────┴──────────────┘
```

Solo visible en escritorio.

---

### Pie de página

```text
Patrocinado por:
Syscom | CT Internacional | PCH | Adobe
```

Muy discreto.

---

### Dashboard

En la pantalla principal.

```text
Herramientas
Documentos
Actividad

Patrocinadores del mes
```

Sin ventanas emergentes.

---

# Nivel 2: Publicidad contextual

Mucho más rentable.

Aparece relacionada con la acción.

Ejemplos:

Usuario comprime PDF.

```text
¿Necesitas firmar PDFs?
Conoce Adobe Acrobat.
```

Usuario convierte Excel.

```text
¿Necesitas análisis avanzado?
Conoce Power BI.
```

Usuario realiza OCR.

```text
Escáneres Epson para OCR masivo.
```

Este tipo suele tener mejor conversión.

---

# Nivel 3: Créditos patrocinados

Es la que más me gusta para DocuCore.

Ejemplo:

```text
Syscom te regala 50 créditos.
```

Botón:

```text
Obtener créditos
```

No obliga al usuario.

---

# Nivel 4: Publicidad recompensada

La que ya comentamos.

Solo cuando el usuario necesita créditos.

---

# Lo que NO pondría

❌ Popups.

❌ Ventanas emergentes.

❌ Anuncios de pantalla completa.

❌ Videos automáticos.

❌ Redirecciones.

❌ Notificaciones falsas.

❌ Anuncios que tapen el documento.

Eso destruiría la confianza en una plataforma profesional.
