# Integracion CT Internacional

## Objetivo

Documentar la futura descarga de informacion de CT Internacional hacia el ERP.

## Estado

```text
PENDIENTE_DE_DEFINIR
```

## Informacion requerida

- Productos.
- Categorias.
- Marcas.
- Existencias.
- Precios/costos.
- Imagenes.
- Fichas tecnicas.
- Atributos tecnicos.
- Productos relacionados.
- Garantias.
- RawData.

## Regla

La integracion solo descarga y transforma datos externos. La decision de publicacion por proyecto vive en `ChannelAssortment`.
